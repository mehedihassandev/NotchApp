## Quick Reference

### Main Components

-   **NotchBarView.swift** - Main container & state management
-   **MediaPlayerManager.swift** - Music detection & control
-   **DashboardView.swift** - Nook tab content
-   **TrayView.swift** - File management tab
-   **MediaDisplayView.swift** - Player UI component

### Key States

-   `isExpanded` - Controls collapsed/expanded view
-   `isHovering` - Tracks mouse hover state
-   `selectedTab` - Active tab (nook/tray)
-   `currentMedia` - Currently playing media info
